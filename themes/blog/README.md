# Blog Theme

Thème de blog moderne et épuré pour ExilonCMS.

## Installation

Ce thème est inclus par défaut avec ExilonCMS. Pour l'activer :

1. Allez dans le panneau d'administration
2. Navigation vers `Apparence > Thèmes`
3. Cliquez sur "Activer" à côté du thème Blog

## Fonctionnalités

- Design épuré moderne
- Support du mode sombre
- Optimisé pour le contenu de blog
- Responsive design
- Personnalisation des couleurs
- Support du logo et favicon

## Personnalisation

Le thème peut être personnalisé via le panneau d'administration :

- Couleur principale
- Logo
- Favicon
- Style de navigation
- Pied de page
